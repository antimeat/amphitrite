Hints

The partition splitting scripts execute for the 00Z and 12Z runs of Auswave each day. Check that the Active Sites are configured correctly using "Config check"

Config check

  • "Config check" should show a green highlighted "Sites config and exclusiions up to date!", otherwise, if sites are highlighted in yellow they should be either added to the configuration file or the exlusion file.
  • the configuration file and the exclusion file both need to be kept up to date
  • update the sites config file using menu item "edit-config"
  • sites config file has the format "forecast_name", "table_name", "partions"
  • update the sites to exclude (eg: land based sites) using menu item "edit-exclude"
  • exlusion file has the format "forecast_name"

Sites list

  • html table of sites from the database linking to api's used for the data retrieval
  • data is the latest available in the database for each particular site

Database check

  • query the database to see the current site list and run_times that are saved

Active sites

  • Ofcast active site list with shift, name, days, lat, lon

Table names

  • Auswave table names and their respective lat lons. Useful for setting up the configuration file

Activate run

  • activate a manual run of the partion splitter choosing individual sites or all
  • by default it will run using the latest available auswave data to split
