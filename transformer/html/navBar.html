Hints & Tricks

  • LOAD data does the following:
    1. pulls guidance data from the data source.
    2. pulls forecast data from Ofcast.
    3. populates wind and hs field boxes from the data source
    4. calulates and populates residual swell field using the formula: take sum of the squares from selected swells in Ofcast, then take the difference between that and the data source guidance. Residual swell field values should then be used to fill the unckecked swell# in Ofcast.
  • Data in the field boxes is always from the data source. It's worth remembering this, as when selecting a previous issue the data field boxes are not from the previous forecast.
  • special-sauce:
    • wind limit: the lower limit used where winds below that limit will not be modified.
    • wind multiplier: that multiplication factor used to multiply wind speeds above the wind limit.
    • hs limit: the lower limit used where hs below that limit will not be modified.
    • hs multiplier: that multiplication factor used to multiply hs above the hs limit.
  • Some selected Ofcast keyboard-shortcuts for pasting data values:
    • SHIFT P: paste wind speed,dir into dialog box for en masse pasting all values.
    • CTRL O: pase individual columns of wind speed,dir each time-step
    • SHIFT O: paste seas/swell amplitude,period,dir into dialog box for en masse pasting all values.
    • CRTL P: paste individual columns of seas/swell amplitude
